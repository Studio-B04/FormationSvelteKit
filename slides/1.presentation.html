1. C'est quoi SvelteKit ?

--- layout: default transition: slide-up ---

🚀 En une phrase :

"SvelteKit est un framework qui sert à construire des applications web modernes avec Svelte"

--- layout: default transition: slide-up ---

Un peu d’histoire…

Quand Svelte a été lancé en 2016, il a très vite attiré l’attention pour son approche radicalement différente : compiler le code à la build pour générer du JavaScript natif, sans runtime encombrant.

Mais voilà : Svelte, à lui seul, ne suffisait pas pour construire une application complète. Il manquait plein de briques qu’on retrouve dans un framework plus "complet" :

Pour créer une vraie app avec Svelte, les développeurs devaient assembler eux-mêmes plusieurs outils, un peu galère, surtout pour des projets sérieux.

--- layout: default transition: slide-up ---

L’arrivée de SvelteKit

C’est dans ce contexte que SvelteKit est annoncé en 2020 comme le successeur de Sapper, un ancien framework maison abandonné.

SvelteKit s’inspire clairement de ce qu’on connaît dans le monde React avec Next.js, ou Vue avec Nuxt, mais avec la légèreté et la philosophie unique de Svelte.
