Création du projet avec l’outil officiel

Sveltekit propose un assistant interactif qui vous guide dans la configuration initiale :

Créez un nouveau dossier, ouvrez le terminal et tapez :

```bash {*}{lines:false} npx sv create . ```

"Which template would you like?"

```bash {*}{lines:false} ● SvelteKit minimal ```

"Add type checking with TypeScript?"

```bash {*}{lines:false} ● No ```

"What would you like to add to your project?"

```bash {*}{lines:false} ◼ prettier ◼ prettier ◼ sveltekit-adapter ```

"Which SvelteKit adapter would you like to use?"

```bash {*}{lines:false} ● static ```

"Which package manager do you want to install dependencies with?"

```bash {*}{lines:false} ● npm ```

Et enfin, lancez le projet avec la commande :

```bash {*}{lines:false} npm run dev ```
